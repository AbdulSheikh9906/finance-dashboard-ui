# Finance Dashboard UI

A clean and interactive finance dashboard to track income, expenses, and financial activity.

## Live Demo

https://elaborate-piroshki-c39124.netlify.app/

## Features

- Summary cards (Total Balance, Income, Expenses)
- Balance trend chart and spending breakdown chart
- Transaction list with search, filter, and sort
- Role-based UI (Viewer can only view, Admin can add/edit/delete)
- Financial insights section
- Responsive design (works on mobile and desktop)

## Tech Used

- React.js
- Tailwind CSS
- Chart.js
- Custom state management

## How to Run

1. Clone the repo
2. Open index.html in your browser

## Role Switching

- **Viewer** - Can only see data
- **Admin** - Can add, edit, and delete transactions

Use the dropdown in the top right to switch roles.
